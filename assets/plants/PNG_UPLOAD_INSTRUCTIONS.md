# PNG plant artwork upload

The final watercolor plant artwork has been split into individual PNGs. Add the PNG files from `nanas-garden-plant-pngs.zip` into this folder with these exact names:

- `bush-early-girl-tomato.png`
- `roma-tomato.png`
- `sweet-million-tomato.png`
- `sun-sugar-tomato.png`
- `jalapeno-pepper.png`
- `slicing-cucumber.png`
- `marigold.png`

The app already references these PNG paths through `PLANT_ASSETS` in `app.js`. Once these files are committed, the live garden will use the watercolor artwork directly.

Keep the existing SVG files as fallbacks/reference art.

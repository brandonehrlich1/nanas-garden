# Plant artwork

These images are curated illustration assets for Nana's Garden plant cards.

The app expects these filenames:

- `bush-early-girl-tomato.jpg`
- `roma-tomato.jpg`
- `sweet-million-tomato.jpg`
- `sun-sugar-tomato.jpg`
- `jalapeno-pepper.jpg`
- `slicing-cucumber.jpg`
- `marigold.jpg`

Replace any file with the same filename to update the artwork without changing app code.
